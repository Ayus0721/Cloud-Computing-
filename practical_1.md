# Introduction to Cloud Platforms ☁️

## 🎯 Objective
To familiarize students with major cloud platforms and their interfaces, including AWS, Azure, and Google Cloud Platform (GCP).

---

## 📚 Overview
This project introduces three leading cloud service providers:
- Amazon Web Services (AWS)
- Microsoft Azure
- Google Cloud Platform (GCP)

Students will create free-tier accounts, explore dashboards, and understand pricing models.

---

## 🛠️ Steps Performed

### a) Create Free Tier Accounts
- Signed up for:
  - AWS Free Tier account
  - Azure Free Account
  - Google Cloud Free Tier account

> Note: Free tiers provide limited usage with no cost under certain limits.

---

### b) Explore Dashboards and Identify Key Services

#### 🔹 AWS Dashboard
- Services explored:
  - EC2 (Virtual Servers)
  - S3 (Storage)
  - Lambda (Serverless Computing)
  - RDS (Database)

#### 🔹 Azure Dashboard
- Services explored:
  - Virtual Machines
  - Blob Storage
  - Azure Functions
  - SQL Database

#### 🔹 GCP Dashboard
- Services explored:
  - Compute Engine
  - Cloud Storage
  - Cloud Functions
  - BigQuery

---

### c) Understand Pricing Calculators

#### 💰 AWS Pricing Calculator
- Estimated costs for EC2, S3, and RDS usage
- URL: https://calculator.aws/

#### 💰 Azure Pricing Calculator
- Estimated costs for Virtual Machines and Storage
- URL: https://azure.microsoft.com/en-in/pricing/calculator/

#### 💰 GCP Pricing Calculator
- Estimated costs for Compute Engine and Cloud Storage
- URL: https://cloud.google.com/products/calculator

---

## 📊 Key Learnings
- Cloud platforms offer similar core services with different naming conventions
- Free tiers are useful for beginners to experiment without cost
- Pricing calculators help estimate and manage cloud expenses

---

## ✅ Conclusion
This exercise provided hands-on exposure to cloud platforms, their interfaces, and pricing models. Understanding these basics is essential for working with cloud technologies.
